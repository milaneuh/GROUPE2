Définition du laguage Javascript

Javascript ou "JS", est un language de programmation compilé. Le Javascript est surtout connus comme un language de script pour les pages webs. Nonobstant, beaucoup d'environnenment qui ne sont pas liés au web l'utilise. Tel que Node.js, Apache CouchDB ou même Adobe Acrobat.
