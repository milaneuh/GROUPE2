Introduction

Javascript ou "JS", est un language de programmation compilé. Le Javascript est surtout connus comme un language de script pour les pages webs. Nonobstant, beaucoup d'environnenment qui ne sont pas liés au web l'utilise. Tel que Node.js, Apache CouchDB ou même Adobe Acrobat.
Historiquement, le JavaScript fût le premier langage de script pour le Web. Le JavaScript permet au client d'executer des commande. (C'est à dire au niveau du navigateur et non du serveur web)

Contrairement au language Java (Avec lequel le JS est souvent confondu) le JavaScript n'a pas besoins de compilateur, ce qui rend le JavaScript fortement dépendant de la page web dans laquelle il est incorporé

JavaScript et Java

Il ne faut pas confondre le JavaScript et le Java. Contrairement au Java. Le JS est codé directement dans la page HTML, c'est un langage qui ne permet aucune confidentialité au niveau des codes (Ceux ci sont facilement visibles)

JavaScript Java
Langage interprété Langage pseudo-compilé
Code intégré au HTML Code à part du document HTML, appelé à partir de la page
Langage peu typé Langage fortement typé
Liaisons dynamiques: les références des objets sont vérifiées au chargement Liaisons statiques: les objets doivent exister au chargement (compilation)
Accessibilité du code Confidentialité du code
Sûr: ne peut pas écrire sur le disque dur Sûr: ne peut pas écrire sur le disque dur
