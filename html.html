Présentation du langage HTML

Le HTML a été imaginé dans les années 1990 par Tim Berners-Lee. HTML est un acronyme pris de l'anglais HyperText Markup Language, ce qui signifie en français langage de balisage d'hypertexte celui désigne un type de langage informatique. Plus précisément, le HTML n'est pas un langage dit de programmation mais, un langage de balisage. Le HTML est un code permettant de structurer les pages WEB et leurs contenus.

Dans sa première version seul les éléments ne comprennent que le titre du document, les hyperliens, la structuration du texte en titres, sous-titres, listes ou texte brut, et un mécanisme rudimentaire de recherche par index.

En 1993, avec l'apparition du navigateur "NCSA Mosaic", les développeurs ont vu une évolution majeure en pouvant intégrer des images (GIF ou XBM) directement sur les pages WEB, ce qui les rends plus attractives.

L'année suivante, est apparu le navigateur "Netscape Navigator" qui a permis des présentations avec des styles telles que le centrage ou le clignotement du texte. Ces nouveaux styles ont donné plus d'impacts visuels aux pages WEB.

Les différences entre le HTML et le HTML5

Lorsque l’on parle de conception de sites Web, on entends souvent parler de “balises“. On se pose souvent la question concernant la signification de ce mot et la différence entre ce que l’on appelle le “code“. Ce que l’on appelle le balisage est conçu afin de traiter, définir et présenter du texte. Le langage HTML est le langage le plus utilisé actuellement. Cependant, une mise à jour de ce dernier à eu lieu, il y a quelques années, il s’agit du HTML5. Cet article va vous permettre de connaitre la différence entre HTML et HTML5.
